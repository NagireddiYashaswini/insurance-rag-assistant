# Insurance RAG Assistant

AI-powered insurance document question-answering system using RAG architecture.

## Tech Stack

* FastAPI
* LangChain
* ChromaDB
* Hugging Face Embeddings

## Features

* Semantic document retrieval
* Insurance policy Q&A
* BM25 + Vector Search
* FastAPI backend

## Run Locally

pip install -r requirements.txt
uvicorn app:app --reload
